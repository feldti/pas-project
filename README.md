# pas-project
files for pas-project support under Gemstone/S

* pas_install_env.sh - Installation Script for PAS folder for a development/production environment under Linux
* pum-binaries - folder with the latest binaries of the PUM modelling tool/code generator tool
* plantuml - folder with supporting tools for PUM
* pas_runtime - folder with runtime libraries for pas development
* pas_docu - folder with HTML playout of the PAS documentation
* pasdevkit_help - Help & Manual sources of the docu
