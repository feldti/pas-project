Konfiguration der einzelnen Maschine

Hier ein Beispiel aus dem mitgelieferten Template. So wird eine Maschine konfiguriert. Neben Java, .Net Runtime werden vor allem die Dateisysteme des HOSTS an bestimmten Stellen eingebunden bzw. die Installation so abgeändert, das Daten im externen Dateisystem abgelegt werden. In diesem Beispiel werden alle Datenbank-Dateien der stones ins HOST-Dateisystem geschrieben. Die zweite Einbindung dient dazu, um externe Daten der VM zur Verfügung zu stellen. Das als exemplarische Nutzung der Möglichkeiten.
