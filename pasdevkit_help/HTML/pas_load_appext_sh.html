pas_load_appext.sh

Beschreibung

Dieses Skript lädt die Anwendungslogik. Diese Skript wird Informationen aus der projektspezifischen "credentials.sh" benötigen.  

Das Skript lädt Code aus Monticello-Paketen und das sind in der Regel

•(A) - Eigenständige Klassen in eigenen Paketen (Klassen und Methods)

•(B) - Erweiterungen der Modell-Klassen (Methods)

Die Reihenfolge beim Laden sollte unbedingt eingehalten werden. Die Erweiterungen der Modell-Klassen (B) sollen immer zuletzt geladen werden, weil diese auch den API-Code beinhaltet und auf alles zugreifen dürfen.

Dieses Skript wird auch genutzt, wenn man BugFixes (des Codes) beim Kunden einspielen möchte.  Das Skript soll keine Dateninitialisierungen durchgeführt werden - nur Laden.
